# AI Wound Severity Detection

An AI-powered web application that analyzes wound images and predicts the wound type and severity.

## Features
- Upload wound image
- Detect wound category
- Predict severity level
- Provide basic suggestions
- User-friendly web interface

## Technologies Used
- Python
- Flask
- TensorFlow / Keras
- HTML
- CSS

## How to Run

```bash
pip install -r requirements.txt
python app.py
```
