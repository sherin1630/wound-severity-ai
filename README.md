# 🩹 AI Wound Severity Detection System

An AI-powered web application that analyzes wound images and predicts the wound type and severity level using machine learning techniques.

## 🚀 Features

* Upload wound images through a simple web interface
* Detect wound category automatically
* Estimate wound severity level
* Display wound percentage analysis
* Provide basic recommendations and suggestions
* Interactive visualization using charts

## 🛠️ Technologies Used

* Python
* Flask
* TensorFlow / Keras
* HTML5
* CSS3
* JavaScript
* Matplotlib

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/sherin1630/wound-severity-ai.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open in browser

```text
http://127.0.0.1:5000
```

## 🎯 Project Goal

The objective of this project is to assist users by providing an AI-based preliminary assessment of wound severity from uploaded images. This system is intended for educational and research purposes and should not replace professional medical diagnosis.
